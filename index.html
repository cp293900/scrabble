Apocript

Objective:　 To spell words with the letters provided.
Player: Solo game
Timer: 1 minute
Dice: 2

1. The cards are in rows and columns.

2. Press the dice. One minute timer.

3. Create a word or words with the cards laying down facing up.
Currently, the user has only one option to create one word.

4. Point system: Each letter has a point. Each point is based on the number of possibilities – level of difficulty for each letter: the greater the difficulty the higher the point.

5. Five rounds. The points are accumulated.

Note: The Wild Card can be any letter the player chooses.
